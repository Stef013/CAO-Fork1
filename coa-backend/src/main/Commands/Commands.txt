COMMANDS FOR THE API & COMMUNICATION THROUGH TERMINAL

-Start API dev:
mvnw compile quarkus:dev

-Start Tests:
mvnw Test

-Send request:
curl -w "\n" [Path]
curl -w "\n" http://localhost:8080/hello/greeting/quarkus

-Create package:
mvn package

-Run java file:
java -jar target/getting-started-1.0.0-SNAPSHOT-runner.jar
